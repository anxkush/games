# Snake and Ladder Game

A fun interactive Snake and Ladder game built with HTML, CSS, and JavaScript. Play against another player on the same computer!

## Features

- Two-player gameplay
- Animated dice rolls
- Visual board representation (10x10 grid)
- Snakes and Ladders that move players automatically
- Turn-based gameplay
- Win detection

## How to Play

1. Open `GAME.HTML` in your web browser
2. Players take turns rolling the dice
3. Click the "Roll Dice" button to move
4. Snakes send you down, Ladders move you up
5. First player to reach square 100 wins!

## Game Rules

- Players start at square 1
- Each turn, roll the dice to move forward
- If you land on a ladder, climb up
- If you land on a snake, slide down
- You cannot move beyond square 100
- If your roll would take you past 100, you stay in place

## Files

- `GAME.HTML` - Main game HTML file

- **HTML5** - Game structure
- **CSS3** - Styling and layout

Works in all modern browsers that support HTML5, CSS3, and ES6 JavaScript.
